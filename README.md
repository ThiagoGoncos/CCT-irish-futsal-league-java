# Irish Futsal League Console Program

## Project Overview

This project was developed as part of the **Bachelor of Science in Computing and Information Technology** at **CCT College Dublin**.

The objective of this small Java project is to display information about clubs participating in the **AUL Futsal League for the 2023/2024 season**.

The program is a simple **console-based Java application** that prints a list of futsal teams in different divisions.

The project demonstrates the use of the **Java main method, console output, and basic program structure**.

---

## Program Functionality

When the program runs, it displays information about futsal clubs competing in different divisions of the **AUL Futsal League 2023/2024 season**.

The program outputs three sections:

1. AUL Futsal Premier Division clubs
2. AUL Futsal 2nd Division clubs
3. Traditional clubs that are not participating in the 2023/2024 season

The output is printed using standard Java console output commands.

---

## AUL Futsal Premier Division (2023/2024)

The program lists the clubs competing in the Premier Division:

- Blue Magic FC  
- Leprechaun FC  (Leinster Futsal now)
- Pro Brazil FC  
- Clontarf Vikings FC  
- St Itas FC  
- Croata Dublin FC  

These teams represent the top division of the league.

---

## AUL Futsal 2nd Division (2023/2024)

The second section of the program prints the clubs participating in the 2nd Division:

- Blue Magic B  
- Old Boys FC  
- DCU Futsal  
- Moldova Boys FC  
- Bohemian Futsal  

These teams compete for promotion to the higher division.

---

## Traditional Clubs Not Participating (2023/2024)

The program also lists several traditional clubs that are not participating in the AUL season for that year.

These include:

- Real Transylvania FC  
- Puskas Hagi Ashbourne FC  
- Dublin City FC  
- Path2Pro FC  
- Ballymun FC  

This section highlights historical clubs that are currently inactive in the competition.
